## Install

```
$ npm install -g yarn
```

```
$ yarn bin
```

```
$ yarn init
```

```
$ yarn
```

```
$ yarn add
```

```
$ yarn add --dev
```

```
$ yarn list
```

```bash
$ yarn global add create-react-app --prefix /usr/local
# the `create-react-app` command is now available globally:
$ which create-react-app
$ /usr/local/bin/create-react-app
$ create-react-app
```
